Instructions

  1. Open Browser Console: Press F12 and go to Console tab
  2. Check Debug Info: Click "Refresh Debug Info" to see what's loaded
  3. Add Products: Go to admin.html, login, and add some products
  4. Refresh: Come back here and click "Refresh Products"
  5. Test Functions: Try these in console:
    • window.productDisplay.products - See all products
    • window.productDisplay.loadProducts() - Reload products
    • window.refreshProducts() - Refresh display
